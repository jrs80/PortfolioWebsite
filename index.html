Background

After a career in law and academia, I decided I anted to do something more enjoyable with my career. I had enjoyed computer programming as a hobby and had taught myself C and C++hobby, so when I decided to make a career change, it seemed obvious that the next programming language I should learn was C#. I took a bootcamp course through TrueCoders. Their C#/.NET program is fantastic. I learned much more than C# syntax: principles of OOP, TDD, and debugging; version control wth Git; database manipulation with SQL (MySQL) and ORMs (Dapper); how to use APIs & JSON; and using HTML/CSS/JavaScript to design web pages (like this one, which I designed from scratch with Bootstrap).

Sample Projects

TheraBase is an ASP.NET Core MVC project that pulls SQL data from a mock mental-healthcare provider database, allowing users to find a provider, book appointments, etc. The database was created with MySQL. The project is mainly written in C#; the user interface utilizes HTML/Razor/CSS and JavaScript. It is fully CRUD-functional, and implements an API for the calendering system. I am planning to add more features to this project, such as user authentication, the ability to add/remove practice areas, search functionality, etc.
